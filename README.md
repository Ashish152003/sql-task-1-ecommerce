# E-commerce Database Schema

## Description
This is an E-commerce database schema designed using MySQL.

## Tables:
- Customer
- Category
- Product
- Orders
- Order_Item
- Payment

## How to Run:
- Open `create_schema.sql` in MySQL Workbench
- Execute the script to create the database and tables
